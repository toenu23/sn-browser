DIY package

Make sure you have your NRS or NXT wallet running.

Windows users:

To run in standalone mode make sure you have java 1.8 installed (https://java.com/ru/download/) and execute run.bat.
Or run the core with run_core.bat. Core requires java 1.7 or newer. Now go to the folder "site" and open index.html.
Internet Explorer not supported.

Linux users:

To run in standalone mode make sure you have oracle java 1.8 installed.
For Ubuntu users:

sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java8-installer

Check if your run.sh has execution privileges:

chmod +x run.sh

Execute run.sh:

./run.sh

Or run the core with run_core.sh. Core requires java 1.7 or newer.
Check if your run_core.sh has execution privileges:

chmod +x run_core.sh

Execute run_core.sh:

./run_core.sh

Now go to the folder "site" and open index.html.
